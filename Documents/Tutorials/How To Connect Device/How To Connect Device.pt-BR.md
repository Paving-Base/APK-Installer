# Como conectar o dispositivo
#### Usando o USB
1. Certifique-se de que o Instalador APK foi atualizado para [a versão mais recente](https://www.microsoft.com/store/productId/9P2JFQ43FPPG "APK Installer")
2. Ative o Modo de Desenvolvedor e abra a Depuração USB ![Modo de Desenvolvedor](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Screenshot_20221002-172252.jpg)
3. Vá para a página de configurações do Instalador APK e selecione seu dispositivo ![Página de configurações](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Snipaste_2022-10-02_17-37-30.png)
4. Volte e continue a instalação ![Continuar a instalação](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Snipaste_2022-10-02_17-34-04.png)
#### Usando o Wi-Fi
1. Certifique-se de que o Instalador APK foi atualizado para [a versão mais recente](https://www.microsoft.com/store/productId/9P2JFQ43FPPG "APK Installer")
2. Ative o Modo de Desenvolvedor e abra a Depuração por Wi-Fi ![Modo de Desenvolvedor](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Screenshot_20221002-174001.jpg)
3. Anote o endereço IP do seu dispositivo ![Endereço IP](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Screenshot_20221002-174200.jpg)
3. Vá para a página de configurações do Instalador APK, conecte e selecione seu dispositivo ![Página de configurações](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Snipaste_2022-10-02_17-46-28.png)
4. Volte e continue a instalação![Continuar a instalação](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Snipaste_2022-10-02_17-34-04.png)
