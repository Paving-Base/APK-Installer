# How To Connect Device
#### Usando USB
1. Certifique-se de que o instalador do APK foi atualizado para [a versão mais recente](https://www.microsoft.com/store/productId/9P2JFQ43FPPG "APK Installer")sim
2. Ative o Modo de Desenvolvedor e abra a Depuração USB ![Modo de Desenvolvedor ](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Screenshot_20221002-172252.jpg)
3. vá para a página de configurações e selecione o seu dispositivo ![Página de configurações1](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Snipaste_2022-10-02_17-37-30.png)
4. Go back, resume installation ![Continuar a instalaçãosim](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Snipaste_2022-10-02_17-34-04.png)
#### Usando wifi
1. Please make sure APK Installer has been updated to [the latest version](https://www.microsoft.com/store/productId/9P2JFQ43FPPG "APK Installer")
2. Ative o Modo de Desenvolvedor e abra a Depuração USB ![Modo de Desenvolvedor sim](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Screenshot_20221002-174001.jpg)
3. Remember the IP address of your device ![Endereço IPsu](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Screenshot_20221002-174200.jpg)
3. Vá para a pagina de configurações, conecte e selecione o seu dispositivo ![Página de configurações
](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Snipaste_2022-10-02_17-46-28.png)
4. Voltar reseta a installação![Não consigo instalar](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Snipaste_2022-10-02_17-34-04.png)
