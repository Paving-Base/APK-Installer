# Cách kết nối thiết bị
#### Thông qua USB
1. Hãy đảm bảo Trình Cài Đặt APK đã được cập nhật lên [phiên bản mới nhất](https://www.microsoft.com/store/productId/9P2JFQ43FPPG "APK Installer")
2. Bật Chế độ Nhà phát triển, mở Gỡ lỗi USB![Chế độ Nhà phát triển](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Screenshot_20221002-172252.jpg)
3. Vào trang Cài đặt của Trình cài đặt APK, chọn thiết bị của bạn![Trang Cài đặt](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Snipaste_2022-10-02_17-37-30.png)
4. Go back, resume installation ![Resume Installation](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Snipaste_2022-10-02_17-34-04.png)
#### Using Wifi
1. Please make sure APK Installer has been updated to [the latest version](https://www.microsoft.com/store/productId/9P2JFQ43FPPG "APK Installer")
2. Bật Chế độ nhà phát triển, mở gỡ lỗi từ xa![Developer Mode](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Screenshot_20221002-174001.jpg)
3. Remember the IP address of your device ![IP address](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20Device/Images/Screenshot_20221002-174200.jpg)
