# איך מתחברים ל WSA
1. Please make sure APK Installer has been updated to the [newest version](https://www.microsoft.com/store/productId/9P2JFQ43FPPG "APK Installer")
2. Please make sure you have installed and running [Windows Subsystem for Android™](https://www.microsoft.com/store/productId/9P3395VX91NR)
3. הפעל את מצב מפתחים, ודא שכתובת ה-IP מוצגת![מצב פיתוח](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20WSA/Images/Snipaste_2022-10-02_17-08-37.png)
4. פתח מחדש את מתקין APK, המשך את ההתקנה![המשך התקנה](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20WSA/Images/Snipaste_2021-10-22_15-10-06.png)
