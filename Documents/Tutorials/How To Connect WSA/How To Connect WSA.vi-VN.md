# Cách kết nối WSA
1. Please make sure APK Installer has been updated to the [newest version](https://www.microsoft.com/store/productId/9P2JFQ43FPPG "APK Installer")
2. Please make sure you have installed and running [Windows Subsystem for Android™](https://www.microsoft.com/store/productId/9P3395VX91NR)
3. Bật Chế độ nhà phát triển, đảm bảo địa chỉ IP được hiển thị ![Chế độ Nhà Phát Triển](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20WSA/Images/Snipaste_2022-10-02_17-08-37.png)
4. Mở lại Trình Cài Đặt APK, tiếp tục cài đặt ![Tiếp tục Cài đặt](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20WSA/Images/Snipaste_2021-10-22_15-10-06.png)
