# Jak połączyć się z WSA
1. Please make sure APK Installer has been updated to [the latest version](https://www.microsoft.com/store/productId/9P2JFQ43FPPG "APK Installer")
2. Please make sure you have installed and running [Windows Subsystem for Android™](https://www.microsoft.com/store/productId/9P3395VX91NR)
3. Włącz tryb dewelopera, upewnij się, że adres IP jest wyświetlany ![Tryb Dewelopera](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20WSA/Images/Snipaste_2022-10-02_19-02-09.png)
4. Otwórz ponownie Instalator APK, wznów instalację ![Wznów instalację](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20WSA/Images/Snipaste_2022-10-02_17-34-04.png)
