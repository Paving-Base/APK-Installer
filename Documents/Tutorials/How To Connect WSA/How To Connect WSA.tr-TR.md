# WSA'ya Nasıl Bağlanılır
1. Please make sure APK Installer has been updated to the [newest version](https://www.microsoft.com/store/productId/9P2JFQ43FPPG "APK Installer")
2. Please make sure you have installed and running [Windows Subsystem for Android™](https://www.microsoft.com/store/productId/9P3395VX91NR)
3. Geliştirici Modu'nu açın, IP adresinin gözüktüğünden emin olun![Geliştirici Modu](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20WSA/Images/Snipaste_2022-10-02_17-08-37.png)
4. APK Installer'ı yeniden açın, yüklemeye devam edin![Yüklemeyi Sürdür](https://raw.githubusercontent.com/Paving-Base/APK-Installer/screenshots/Documents/Tutorials/How%20To%20Connect%20WSA/Images/Snipaste_2021-10-22_15-10-06.png)
